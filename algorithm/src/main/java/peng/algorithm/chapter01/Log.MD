--1.1 算法和数据结构
----什么是算法：（引入高斯算法）
        算法是一系列的程序指令，用于处理特定的运算和逻辑问题
    算法有高效的也有拙劣的，衡量算法的指标：
        时间复杂度和空间复杂度
    应用范围：
     运算
     查找
     排序
     最优决策
----什么是数据结构
    数据结构有那些组成：
      线性结构
      树
      图
      其他数据结构
