什么是数组
  有限个相同类型的变量所组成的有序集合
  顺序存储

  数组的优劣势：
    查找快
    插入/删除影响效率
=====================
什么是链表
    物理上非连续，非顺序的数据结构，由若干节点组成
=====================
栈
    是一种线性数据结构
    先进后出（FILO）
    最开始进入的元素叫做栈底（bottom）
    最后进入的元素叫做栈顶（top）
    可以用数组实现，也可以用链表实现
队列
    是一种线性数据结构
    先进先出（FIFO)
    队列的出口叫队头（front）
    队列的入口叫队尾（rear）
注：代码循环队列
